# Data dictionary

Dataset: GBD Autism Spectrum Disorder prevalence (OECD cohort)

Source:
IHME Global Burden of Disease Results Tool  
https://vizhub.healthdata.org/gbd-results/

---

## Variables

location_name  
Country name.

sex_name  
Sex category (Hombre / Mujer in the raw dataset).

age_name  
Age group.

year  
Observation year.

val  
Prevalence rate of autism spectrum disorders.

upper  
Upper bound of the uncertainty interval.

lower  
Lower bound of the uncertainty interval.

measure_name  
Measure used in the dataset (Prevalencia).

metric_name  
Metric used (Tasa).
